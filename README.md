## （規劃中，未完成）搭配 AI 一起學

如果你在讀書的過程中遇到看不懂的段落、想要練習題，或寫的程式跑不起來，可以下載 [**`rust-book-src.zip`**](https://andyshiue.github.io/rust-book/rust-book-src.zip)，整包丟給能讀檔的 AI。記得告訴 AI 兩件事：

1. **「請先閱讀壓縮檔裡的 `GUIDE.md` 再回答我」**——那是寫給 AI 看的補充指令，會讓 AI 的回答貼近這本書的口吻、不會塞給你還沒學過的概念。
2. 你目前讀到第幾章第幾集。

壓縮檔包含 `GUIDE.md`、`SUMMARY.md` 與全部章節原始檔，每次本書更新時 zip 也會自動跟著更新。
