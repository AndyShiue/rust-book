# 第一章第 5 集：算術運算子

## 本集目標
學會在 Rust 裡做加減乘除和取餘數。

## 正文

今天來學數學！別怕，就是加減乘除而已。

### 基本四則運算

先建立兩個變數：

```rust
fn main() {
    let a = 10;
    let b = 3;

    println!("{} + {} = {}", a, b, a + b);
    println!("{} - {} = {}", a, b, a - b);
    println!("{} * {} = {}", a, b, a * b);
    println!("{} / {} = {}", a, b, a / b);
    println!("{} % {} = {}", a, b, a % b);
}
```

跑起來會印出：

```
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3
10 % 3 = 1
```

### 等等，10 / 3 怎麼是 3？

好問題！因為 `a` 和 `b` 都是整數，所以 Rust 做的是**整數除法**，小數點後面直接砍掉。10 除以 3 等於 3.333...，砍掉小數就是 3。

### % 是什麼？

`%` 叫做**取餘數**（模數運算）。10 除以 3 等於 3 餘 1，所以 `10 % 3` 就是 `1`。

你可以想成：「10 裡面有幾個 3？有 3 個，然後剩下 1。」那個剩下的就是餘數。

### 多個 {} 的用法

你有注意到嗎？我們在 `println!` 裡面放了三個 `{}`：

```rust
println!("{} + {} = {}", a, b, a + b);
```

Rust 會按照順序把值填進去：
- 第一個 `{}` → `a` 的值（10）
- 第二個 `{}` → `b` 的值（3）
- 第三個 `{}` → `a + b` 的值（13）

幾個 `{}` 就對應後面幾個值，順序要對上。

## 重點整理
- 五個算術運算子：`+`（加）、`-`（減）、`*`（乘）、`/`（除）、`%`（取餘數）
- 整數除法會直接捨去小數部分（`10 / 3` 是 `3` 不是 `3.333`）
- `%` 取餘數：`10 % 3` 就是 10 除以 3 剩下的 `1`
- `println!` 裡可以放多個 `{}`，按順序對應後面的值
