# 第一章第 7 集：比較運算子

## 本集目標
學會用比較運算子來比大小、判斷相不相等。

## 正文

到目前為止我們都在做數學運算，但程式設計裡還有另一種很重要的運算——**比較**。

比較的結果不是數字，而是 `true`（對）或 `false`（錯）。

### == 等於

```rust
fn main() {
    println!("{}", 5 == 5);
}
```

印出 `true`。5 等於 5 嗎？對，所以是 `true`。

注意喔，是**兩個等號** `==`，不是一個。一個等號 `=` 是拿來給變數賦值的（`let x = 5`），兩個等號 `==` 才是拿來比較的。

### != 不等於

```rust
fn main() {
    println!("{}", 5 != 3);
}
```

印出 `true`。5 不等於 3 嗎？對。

### < 小於

```rust
fn main() {
    println!("{}", 3 < 5);
}
```

印出 `true`。3 小於 5。

### > 大於

```rust
fn main() {
    println!("{}", 10 > 7);
}
```

印出 `true`。10 大於 7。

### <= 小於等於

```rust
fn main() {
    println!("{}", 5 <= 5);
}
```

印出 `true`。5 小於或等於 5 嗎？等於的話也算，所以是 `true`。

### >= 大於等於

```rust
fn main() {
    println!("{}", 8 >= 10);
}
```

印出 `false`。8 大於或等於 10 嗎？不是。

### 一覽表

| 運算子 | 意思 | 範例 | 結果 |
|--------|------|------|------|
| `==` | 等於 | `5 == 5` | `true` |
| `!=` | 不等於 | `5 != 3` | `true` |
| `<` | 小於 | `3 < 5` | `true` |
| `>` | 大於 | `10 > 7` | `true` |
| `<=` | 小於等於 | `5 <= 5` | `true` |
| `>=` | 大於等於 | `8 >= 10` | `false` |

## 重點整理
- 六個比較運算子：`==`、`!=`、`<`、`>`、`<=`、`>=`
- 比較的結果是 `true`（對）或 `false`（錯）
- `==`（兩個等號）是比較，`=`（一個等號）是賦值，別搞混
